UALandingPageBuilderExtender

Objective-C

@protocol UALandingPageBuilderExtender <NSObject>

Swift

protocol UALandingPageBuilderExtender : NSObjectProtocol

A protocol for extending the default conversion between legacy landing pages and scheduled HTML message landing pages.

  • -extendScheduleBuilder:

    Extends the in-app schedule builder converted from a legacy landing page. Use this method to make use of the default conversion with minor overrides as needed.

    Declaration

    Objective-C

    - (void)extendScheduleBuilder:(nonnull UAScheduleBuilder *)builder;

    Swift

    optional func extend(_ builder: UAScheduleBuilder)

    Parameters

    builder

    The automatically converted in-app schedule builder.
  • -extendMessageBuilder:

    Extends the in-app message builder converted from a legacy landing page action. Use this method to make use of the automatic conversion process with minor overrides as needed.

    - (void)extendMessageBuilder:(nonnull UAInAppMessageBuilder *)builder;

    Swift
