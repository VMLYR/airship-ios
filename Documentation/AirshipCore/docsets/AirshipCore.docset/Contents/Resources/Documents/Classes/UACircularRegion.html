UACircularRegion

Objective-C

@interface UACircularRegion : NSObject

Swift

class UACircularRegion : NSObject

A UACircularRegion defines a circular region with a radius, and latitude and longitude from its center.

Circular Region Factory

  • +circularRegionWithRadius:latitude:longitude:

    Factory method for creating a circular region.

    Declaration

    Objective-C

    + (nullable instancetype)circularRegionWithRadius:(nonnull NSNumber *)radius
                                         latitude:(nonnull NSNumber *)latitude
                                        longitude:(nonnull NSNumber *)longitude;

    Swift

    convenience init?(radius: NSNumber, latitude: NSNumber, longitude: NSNumber)

    Parameters

    radius

    The radius of the circular region in meters.
    latitude

    The latitude of the circular region’s center point in degress.
    longitude

    The longitude of the circular region’s center point in degrees.

    Return Value

    Circular region object or nil if error occurs
